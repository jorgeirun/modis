
COMPILATION
===========

	make (tout simplement)


UTILISATION
===========

La bibliothèque libicaf77hdf.a générée dans le répertoire lib contient un
certain nombre de fonctions permettant de manipuler de manière plus ou moins
générique les SDS d'un fichier HDF.

Pour l'utiliser, il suffit de compiler le programme appelant avec les options

	 -L<libdir> -licaf77hdf

où <libdir> représente le chemin du répertoire ou se trouve cette bibliothèque.
